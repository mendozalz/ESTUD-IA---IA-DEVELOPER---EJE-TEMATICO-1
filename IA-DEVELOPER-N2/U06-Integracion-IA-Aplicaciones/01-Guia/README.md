# U06 - Integración de IA en aplicaciones reales

## Descripción breve

Despliegue de modelos como servicio (Flask/FastAPI) e integración con un front-end o cliente básico, cuidando contratos de entrada/salida y robustez.

## Objetivos de aprendizaje específicos

- Desarrollar aplicaciones prácticas con IA integrada.
- Desplegar un modelo como API de predicción.
- Integrar un cliente para consumir el servicio y validar casos reales.

## Temas clave a cubrir

- FastAPI/Flask: endpoints, validación, manejo de errores.
- Carga del modelo, performance básico y warmup.
- Contratos de API (schemas) y versionado.
- Seguridad básica (secretos, límites de payload).

## Lecciones sugeridas

- Lección 1: Diseño del contrato de inferencia.
- Lección 2: Implementación de API con FastAPI.
- Lección 3: Cliente (front-end básico o script).
- Lección 4: Pruebas de integración y hardening.

## Proyecto práctico sugerido

Servicio de predicción con FastAPI + cliente de consumo + casos de prueba.
