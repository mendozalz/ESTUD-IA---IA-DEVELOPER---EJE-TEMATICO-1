# U08 - Proyecto integrador: Aplicación IA automatizada

## Descripción breve

Proyecto end-to-end: definición del problema, pipeline automatizado, entrenamiento/tuning, despliegue como API, monitoreo básico y presentación.

## Objetivos de aprendizaje específicos

- Aplicar conocimientos en un proyecto completo para automatización.
- Diseñar un backlog técnico y plan de sprints/entregables.
- Entregar una demo funcional con reporte técnico (métricas, decisiones, riesgos).

## Temas clave a cubrir

- Alcance, métricas, riesgos y criterios de éxito.
- Arquitectura del sistema (datos → modelo → API → cliente).
- Calidad y robustez (validación, errores, pruebas básicas).
- Presentación técnica y retroalimentación.

## Lecciones sugeridas

- Lección 1: Kickoff y diseño.
- Lección 2: Pipeline y baseline.
- Lección 3: Mejoras y tuning.
- Lección 4: Despliegue y demo.
- Lección 5: Cierre, retro y próximos pasos.

## Proyecto práctico sugerido

Aplicación completa con pipeline automatizado + modelo exportado + API + cliente + reporte final.
