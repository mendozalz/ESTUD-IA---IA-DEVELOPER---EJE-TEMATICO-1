# U04 - Desarrollo de aplicaciones prácticas de clasificación

## Descripción breve

Implementación de soluciones prácticas de IA para clasificación (CNNs) aplicadas a imágenes y/o texto, con evaluación y análisis de errores.

## Objetivos de aprendizaje específicos

- Implementar soluciones prácticas de IA para clasificación.
- Entrenar un modelo de clasificación avanzado (CNN / transfer learning).
- Evaluar el rendimiento y analizar errores para iterar mejoras.

## Temas clave a cubrir

- CNNs: convolución, pooling, data augmentation.
- Transfer learning y fine-tuning.
- Métricas y matriz de confusión.
- Preparación de datos y control de data leakage.

## Lecciones sugeridas

- Lección 1: Dataset, partición y augmentations.
- Lección 2: CNN baseline.
- Lección 3: Transfer learning + fine-tuning.
- Lección 4: Reporte de errores por clase y ajustes.

## Proyecto práctico sugerido

Clasificador de imágenes (o texto) con transfer learning, reporte de métricas y análisis de casos fallidos.
