# U09 - English for Tech

## Descripción breve

Inglés técnico para lectura de documentación, escritura de tickets/README y comunicación efectiva en equipos de desarrollo.

## Objetivos de aprendizaje específicos

- Leer documentación técnica y extraer información accionable.
- Escribir comunicación técnica clara (issues, PRs, reportes de experimentos).
- Presentar un resumen técnico breve del proyecto (oral/escrito).

## Temas clave a cubrir

- Vocabulario técnico y verbos de acción.
- Estructura de documentación y troubleshooting.
- Writing: tickets, README, changelog, release notes.
- Speaking: demo y actualización de estado.

## Lecciones sugeridas

- Lección 1: Lectura de docs y estrategias.
- Lección 2: Escritura de issues/PRs.
- Lección 3: README técnico (estructura y claridad).
- Lección 4: Presentación corta del proyecto.

## Proyecto práctico sugerido

README en inglés del proyecto integrador + “technical summary” de 1 página.
