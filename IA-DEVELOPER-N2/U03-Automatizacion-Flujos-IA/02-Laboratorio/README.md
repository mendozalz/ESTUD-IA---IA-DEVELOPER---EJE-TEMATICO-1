# U03 - Laboratorio

## Prácticas sugeridas

- Construir un script `train.py` con argumentos y logging.
- Construir un script `fetch_data.py` para consumir una API.
- Unificar todo en un `run_pipeline.py` o Makefile simple.

## Entregables

- Pipeline ejecutable desde terminal.
- Carpeta `artifacts/` con modelo exportado y métricas.
- Reporte en Markdown del proceso y resultados.
