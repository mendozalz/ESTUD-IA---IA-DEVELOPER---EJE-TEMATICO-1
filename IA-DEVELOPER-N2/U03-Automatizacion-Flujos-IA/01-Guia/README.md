# U03 - Automatización de flujos de trabajo de IA

## Descripción breve

Creación de pipelines de datos, automatización con scripts Python e integración con APIs para ejecutar entrenamiento/evaluación de forma repetible.

## Objetivos de aprendizaje específicos

- Automatizar el ciclo de vida de modelos de IA.
- Diseñar un pipeline (ingesta → validación → entrenamiento → evaluación → artefactos) ejecutable por script.
- Integrar datos/servicios externos mediante APIs, con manejo de errores y límites.

## Temas clave a cubrir

- CLI en Python (parámetros, perfiles de ejecución).
- Logging y manejo de errores.
- Integración con APIs (requests, paginación, timeouts, retries).
- Estructura de proyecto y empaquetado mínimo.

## Lecciones sugeridas

- Lección 1: Diseño del pipeline y contratos de entrada/salida.
- Lección 2: Automatización por CLI y configuración.
- Lección 3: Integración con API y normalización de datos.
- Lección 4: Artefactos: guardar métricas, modelos y reportes.

## Proyecto práctico sugerido

Pipeline por CLI que descarga datos desde API, entrena, evalúa y exporta artefactos (modelo + métricas + reporte).
