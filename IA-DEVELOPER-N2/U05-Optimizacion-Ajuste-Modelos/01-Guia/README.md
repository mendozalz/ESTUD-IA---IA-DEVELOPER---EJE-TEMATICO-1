# U05 - Optimización y ajuste de modelos

## Descripción breve

Optimización de modelos para mejorar rendimiento y generalización mediante regularización, tuning de hiperparámetros y análisis sistemático de resultados.

## Objetivos de aprendizaje específicos

- Optimizar modelos de IA para mejorar su rendimiento.
- Aplicar regularización (dropout, L2, early stopping) para mitigar overfitting.
- Ejecutar tuning de hiperparámetros y seleccionar una configuración final basada en evidencia.

## Temas clave a cubrir

- Curvas de aprendizaje y diagnóstico.
- Regularización: dropout, weight decay/L2, data augmentation.
- Hiperparámetros: LR, batch size, arquitectura, optimizador.
- Comparación y registro experimental.

## Lecciones sugeridas

- Lección 1: Diagnóstico (overfitting/underfitting).
- Lección 2: Regularización y callbacks.
- Lección 3: Búsqueda de hiperparámetros (grid/random).
- Lección 4: Selección final y reporte.

## Proyecto práctico sugerido

Tuning con múltiples corridas y reporte comparativo con decisión final justificada.
