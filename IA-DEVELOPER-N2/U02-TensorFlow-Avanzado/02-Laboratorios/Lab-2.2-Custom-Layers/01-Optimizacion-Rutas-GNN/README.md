# Optimización de Rutas con Graph Neural Networks

## 📋 Descripción

Este ejercicio implementa un sistema de optimización de rutas logísticas utilizando Graph Neural Networks (GNNs) personalizados. El sistema aprende patrones de tráfico y optimiza rutas en tiempo real para flotas de vehículos.

## 🎯 Objetivos

- Implementar capas GNN personalizadas en TensorFlow
- Desarrollar algoritmos de optimización de rutas
- Integrar datos de tráfico en tiempo real
- Evaluar performance del sistema

## 📁 Archivos

- `dynamic_gnn_model.py`: Implementación principal del GNN
- `arbol_problemas.py`: Manejo de problemas y restricciones
- `requirements.txt`: Dependencias del proyecto

## 🚀 Ejecución

```bash
# Instalar dependencias
pip install -r requirements.txt

# Ejecutar optimización
python dynamic_gnn_model.py
```

## 📊 Métricas Esperadas

- Reducción de costos: 15-20%
- Mejora en tiempo de entrega: 10-15%
- Precisión de predicción: >85%
