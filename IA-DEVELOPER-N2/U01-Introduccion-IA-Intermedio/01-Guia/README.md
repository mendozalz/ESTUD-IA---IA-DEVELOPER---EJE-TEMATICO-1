# U01 - Introducción al desarrollo de IA (Intermedio)

## Descripción breve

Revisión de conceptos de IA/ML, flujo de trabajo completo y fundamentos de reproducibilidad para proyectos de IA orientados a desarrollo de software.

## Objetivos de aprendizaje específicos

- Entender el desarrollo de IA para aplicaciones prácticas.
- Diferenciar ML clásico vs. Deep Learning y seleccionar un enfoque adecuado.
- Estructurar un proyecto reproducible (entorno, dependencias, semillas, configuración, experimentos).

## Temas clave a cubrir

- Tipos de aprendizaje y tareas comunes.
- Validación, leakage, partición de datos.
- Reproducibilidad y estructura de proyecto.
- Baselines y comparación de enfoques.

## Lecciones sugeridas

- Lección 1: IA para developers: casos reales y trade-offs.
- Lección 2: Entornos, dependencias y reproducibilidad.
- Lección 3: Métricas, validación y leakage.
- Lección 4: Baseline con scikit-learn vs. baseline con Keras.

## Proyecto práctico sugerido

Baseline doble (scikit-learn + Keras) sobre el mismo dataset con comparación de métricas y decisión técnica.
