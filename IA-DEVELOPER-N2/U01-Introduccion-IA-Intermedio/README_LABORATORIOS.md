# U01 - Laboratorios de Introducción a la IA

## Estructura de Laboratorios

### 🌡️ Laboratorio 1 - Conversor de Temperaturas
- **Carpeta:** `Laboratorio 1 - conversor temperaturas/`
- **Archivo:** `conversor.py`
- **Técnica:** Programación básica con Python
- **Objetivo:** Implementar un conversor de temperaturas entre Celsius, Fahrenheit y Kelvin

#### Características del Laboratorio 1:
- **Programación fundamental**: Variables, funciones, control de flujo
- **Manejo de datos**: Conversión entre diferentes unidades
- **Validación**: Manejo de errores y casos extremos
- **Interfaz de usuario**: Menú interactivo y retroalimentación

### 😊 Laboratorio 2 - Sistema de Reconocimiento de Emociones
- **Carpeta:** `Laboratorio 2 - sistema de reconocimiento de emociones/`
- **Archivo:** `emociones.py`
- **Técnica:** Red Neuronal Convolucional (CNN) con TensorFlow/Keras
- **Objetivo:** Clasificar emociones faciales usando técnicas de Computer Vision

#### Características del Laboratorio 2:
- **Computer Vision**: Procesamiento de imágenes faciales
- **Data Augmentation**: Técnicas de aumento de datos
- **CNN**: Arquitectura convolucional para clasificación
- **Evaluación**: Métricas de clasificación y visualización

### 🌸 Laboratorio 3 - Tres Paradigmas de Machine Learning
- **Carpeta:** `Laboratorio 3 - 3 Ejercicios Supervisado No Supervisado Reforzado/`
- **Archivos:** `Ej1.py`, `Ej2.py`, `Ej3.py`
- **Técnicas:** Random Forest, K-Means, Q-Learning
- **Objetivo:** Comparar los 3 paradigmas principales de ML y adaptarlos al proyecto integrador

#### Metodología del Laboratorio 3:
**Fase 1: Identificación y Ejecución**
- Distinguir y reconocer cuál ejercicio corresponde a aprendizaje supervisado, no supervisado y reforzado
- Ejecutar los 3 prototipos para entender su funcionamiento y diferencias

**Fase 2: Adaptación al Proyecto Integrador**  
- Relacionar cada paradigma con tu proyecto integrador
- Generar 3 prototipos adaptados a tus propuestas específicas
- Crear recursos que aporten directamente a tu proyecto final

## Progresión de Aprendizaje

### Nivel 1: Fundamentos (Laboratorio 1)
- **Conceptos**: Programación básica, algoritmos, estructuras de datos
- **Habilidades**: Lógica de programación, manejo de errores, interfaz de usuario
- **Aplicación**: Herramientas prácticas de conversión

### Nivel 2: Machine Learning Básico (Laboratorio 2)
- **Conceptos**: Redes neuronales, convoluciones, computer vision
- **Habilidades**: TensorFlow/Keras, preprocesamiento de imágenes, data augmentation
- **Aplicación**: Clasificación de imágenes en tiempo real

### Nivel 3: Paradigmas de ML (Laboratorio 3)
- **Conceptos**: Aprendizaje supervisado, no supervisado, por refuerzo
- **Habilidades**: Comparación de algoritmos, adaptación a problemas reales
- **Aplicación**: Selección apropiada de técnicas según el problema

## Evaluación y Rúbrica

### Criterios de Evaluación
1. **Funcionalidad**: El código funciona correctamente y produce los resultados esperados
2. **Calidad del Código**: Estructura limpia, comentarios, nombres descriptivos
3. **Comprensión**: Explicación clara de los conceptos y algoritmos utilizados
4. **Creatividad**: Extensiones y mejoras propuestas
5. **Documentación**: README completos y bien estructurados

### Ponderación
