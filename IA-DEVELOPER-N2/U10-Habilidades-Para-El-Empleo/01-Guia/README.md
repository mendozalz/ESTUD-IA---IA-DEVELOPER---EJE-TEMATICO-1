# U10 - Habilidades para el empleo

## Descripción breve

Habilidades socioemocionales, metodologías ágiles adaptadas a proyectos de IA, y empleabilidad (portafolio, entrevistas, comunicación profesional).

## Objetivos de aprendizaje específicos

- Potenciar empleabilidad y colaboración en proyectos tecnológicos.
- Adaptar metodologías ágiles para proyectos de desarrollo de IA.
- Preparar portafolio técnico con evidencia (código, métricas, demos).

## Temas clave a cubrir

- Trabajo en equipo, roles y liderazgo.
- Agile para IA: experimentación, definición de “hecho”, riesgos.
- Portafolio: estructura, proyectos, README, métricas.
- Preparación de entrevistas y pruebas técnicas.

## Lecciones sugeridas

- Lección 1: Comunicación y trabajo en equipo.
- Lección 2: Agile aplicado a IA.
- Lección 3: Portafolio (GitHub) y storytelling.
- Lección 4: Simulación de entrevista técnica.

## Proyecto práctico sugerido

Portafolio con 2–3 proyectos curados + pitch de 3–5 minutos del proyecto integrador.
